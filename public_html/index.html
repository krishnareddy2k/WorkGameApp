Word Puzzle Game Application :
This application generates a random word and shuffle's it and places the word characters in draggable letter boxes, The player needs to guess/identify the characters and drop them in drop zone letter boxes.

Steps to play the Game:
   1.Register / Sign Up
   2.Login
   3.Go to Games page
   4.Start New Game
   5.Quit Game ( if you want to quit from the current game).
